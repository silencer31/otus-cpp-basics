Для сборки проекта потребуется фреймворк Qt не ниже 5 версии, но не 6. Можно Qt 5.15 например.

Сборка проекта под Linux:
1. Рядом с папкой с проектом создать папку build

2. Перейти в папку build при помощи команды:
cd build

3. Выполнить команду:
qmake-qt5 ../knights_move/knights_move.pro 
Должен появиться Makefile

4. Выполнить команду:
make

Должна начаться сборка проекта, в результате которой появится исполнимый файл knights_move

5. Запустить программу можно командой:
./knights_move


Сборка проекта под Windows 10 ( На 11 не проверял ).
Потребуется фреймворк Qt не ниже 5 версии и MSVC. Qt 6 версии не подходит.

Если установлен Qt Creator, то можно открыть в нём проект, указав на файл knights_move.pro и далее выполнить из меню:
1. Build -> Run qmake
2. Build -> Build Project "knights_move"

Рядом с папкой с проектом появится папка вида:
build-knights_move-Desktop_Qt_5_15_1_MSVC2019_64bit-Release
Точное имя зависит от версии Qt и версии MSVC.

Внутри данной папки появится подпапка release, а в ней исполнимый файл программы knights_move.exe

Другой вариант сборки под Windows из консоли:
1. Рядом с папкой с проектом создать папку build

2. Открыть консоль, поставляемую с MSVC и перейти в созданную ранее папку build.

3. Выполнить команду:
qmake ../knights_move/knights_move.pro -r -spec win32-msvc
Если файл qmake не будет найден, то запустить, указав полный путь вида:
E:\Qt\5.15.1\msvc2019_64\bin\qmake ../knights_move/knights_move.pro -r -spec win32-msvc
Полный путь к qmake файлу зависит от версии Qt и MSVC. И qmake должен относиться именно к 5 версии Qt.

4. Выполнить команду:
E:\Qt\Tools\QtCreator\bin\jom\jom.exe /S /X Build.log -f Makefile.Release
Полный путь до jom.exe зависит от того, куда именно проинсталлирован Qt.

5. Внутри папки build появится подпапка release, а в ней исполнимый файл knights_move.exe


Особенность запука исполнимого файла под Windows.
После сборки под Linux, испонимый файл knights_move запускать можно сразу, но при попытке сделать это под Windows, на экране будет ошибка, что не удается найти некоторые библиотеки Qt.
 Чтобы эту проблему решить сделать нужно следующее:

1. Создать любую отдельную папку, например knights_move_game, и скопировать в неё бинарный файл knights_move.exe  

2. Открыть консоль, которая поставляется вместе с фреймворком Qt. Её можно найти в меню Пуск в разделе с Qt. 
Название вида: Qt 5.15 MSVC 64bit   ( может отличаться в зависимости от версии ).

3. В консоли перейти в новую папку с файлом  knights_move.exe

4. Выполнить команду:
windeployqt .\knights_move.exe

5. Рядом с файлом knights_move.exe появятся файлы различных библиотек, после чего файл можно будет запускать.


Варианты запуска программы.
1. Самое простое это сделать двойной клик по файлу программы. Игра запуститься с полем размером 10 на 10.

2. Тоже самое можно сделать, запустив knights_move или knights_move.exe из консоли.

3. Если есть желание играть на поле другого размера, то есть два варианта.

1) В файле main.cpp явно прописать размер одной стороны поля и пересобрать проект. В верхней части файла main.cpp перечислены 3 варианта размера поля по умолчанию.

2) Запустить игру из консоли, передав параметры:
knights_move -l 5

-l от слова level (т.е. там маленькая эль, а не большая ай).
Последний параметр - цифра, означающая размер одной стороны. Она может быть в диапазоне от 1 до 10 включительно.

Таким образом, запустив программу командой
knights_move -l 5
появится игра с размером поля 5 на 5. 

После появления программы на экране, нажмите кнопку Начать игру.
Программа имеет интуитивно понятный интерфейс.
